# RyAgent Station Setup

One-click install script for Ryan's AI Station on Ubuntu 24.04.

## Services
- llama.cpp (GPU + CPU) - Local LLM inference
- SillyTavern - AI chat frontend
- ComfyUI - Stable Diffusion
- S2S Voice - Realtime voice assistant
- UbuntuConsole - Services dashboard
- Camofox Browser - Anti-detection browser
- Firecrawl - Web scraping engine
- Telegram Bot API
- RustDesk - Remote desktop
- Docker - Container runtime

## Quick Start
```bash
bash setup-ryan-station.sh
```
